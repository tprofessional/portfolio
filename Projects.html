Project: West Point Bridge Design
Course POE (Principles of Engineering)
Duration: ~3 weeks (5 March, 2020 - 20 March, 2020)
Objective: Design a cost-effective and functional bridge which distributes and supports tensile and compression forces.
Type of Project: Group Project

The goal of this project was to use a decision matrix to select and modify a single bridge design using the West Point Bridge Designer software. The final bridge had a budget limit of $300,000 and needed to be able to support weight by distributing tensile and compression forces throughout its members. My primary roles in this project were to develop 2 initial designs, update and write thorough documentation, and improve the final design my partner and I selected. Of the four total initial designs my partner and I came up with, a design I developed, Design 2B, fit our ideal criteria the most. I used a decision matrix to further improve Design 2B so that it would be less costly and better able to distribute tensile and compressive forces. The decision matrix was an important part of this project because it defined the characteristics of the final proposal for our bridge design. This helped me improve my descriptive writing in documentation because we needed to be specific when identifying the characteristics of the bridge we had improved in our final design.

Project: Star Data Reader
Course CSA (Computer Science Applications)
Duration: 1 week (26 April, 2021 - 30 April, 2021)
Objective: Read and process a dataset using Java and the Sinbad library. Include the use of arraylists.
Type of Project: Group Project

This project built on the topics covered in the PLTW DATA Lab and required that we use arraylists and the Sinbad library to read and process data. During this project, I learned how to find data on Kaggle and how to process data using Sinbad and the Java DataSource interface. We used a .csv file and the DataSource interface to load the data. For this project, we also had to incorporate our past knowledge of creating classes and constructors to assign each data entry to a Star object of our own Star class. In our class file, we created accessor, or getter methods to return each attribute of a Star object that was relevant to the purpose of our program. Our program would ultimately ask the user if they wanted to know the information of the hottest or coldest star in the dataset, and would then return the appropriate attributes of that star. To do this, we needed to use our accessor methods to return the temperature of a Star, and then use the DataSource fetchList() method so that we could iterate through an array of all the stars in the dataset and find the one with either the highest or lowest temperaature (depending on which one the user wanted to find).
